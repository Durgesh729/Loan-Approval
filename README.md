## ✨ Overview

This project is an end-to-end **Machine Learning web application** that predicts whether a loan application is likely to be **Approved** or **Rejected** based on applicant financial and credit information.

The application combines a **Random Forest Classifier**, real-time data preprocessing, and a modern **FastAPI** backend to deliver fast, accurate, and probability-based predictions through an intuitive multi-step interface.
